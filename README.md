# Dicom Veiwer

Dicom Viewer is a Python code to view dicom images of CT scan.

## Installation Required

- Python 3.6+ 
- PyQt5

```bash
pip install PyQt5
```
- Pydicom
```bash
pip install Pydicom
```
## Usage

```python
python app.py
```
